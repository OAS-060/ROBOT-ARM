روبوت آلي — لوحة تحكم الذراع 6 محاور
======================================

التعديل الحالي
--------------
1) لا يرسل الأردوينو أي نبضة للسيرفوات عند التشغيل.
2) خرج PCA9685 يبقى معطلاً هاردوير عبر OE حتى يصل أمر مباشر من الصفحة.
3) بعد انتهاء كل أمر يتم تعطيل OE وتحرير السيرفو؛ لذلك لا يبقى ممسكًا بالزاوية ولا يحاول الرجوع إذا تحرك بفعل الوزن أو باليد.
4) لا تتحرك إلا قناة واحدة في كل مرة.
5) أزيلت الحدود الخاصة بكل مفصل، وأصبح المجال العام في الصفحة 0° إلى 180°.

توصيل الأمان الضروري لـ OE
--------------------------
PCA9685 OE  -> Arduino D8
PCA9685 OE  -> مقاومة 10k -> Arduino 5V / PCA VCC

OE فعّال بالمستوى المنخفض:
HIGH = كل مخارج PCA9685 معطلة
LOW  = المخارج مفعلة أثناء الحركة فقط

المقاومة 10k مهمة لأنها تبقي OE مرتفعًا أثناء إقلاع الأردوينو أو إعادة تشغيله، قبل أن يبدأ الكود.
لا توصل OE إلى GND في هذا الإصدار.

توصيل I2C
---------
Arduino 5V  -> PCA9685 VCC
Arduino GND -> PCA9685 GND
Arduino A4  -> PCA9685 SDA
Arduino A5  -> PCA9685 SCL

تغذية السيرفوات
---------------
مزود خارجي 5–6V موجب -> PCA9685 V+
مزود خارجي GND       -> PCA9685 GND
ويجب توحيد GND المزود مع GND الأردوينو.

تنبيه مهم جدًا
--------------
لا توصل أو تفصل فيشة السيرفو بينما V+ شغالة. التوصيل الحار قد يسبب حركة لحظية بسبب ترتيب ملامسة أسلاك الإشارة والطاقة، حتى لو كان OE معطلاً. افصل تغذية السيرفوات أولًا، وصّل جميع الفيش، ثم شغّل الطاقة.

حدود الدوران
------------
MG996R وMG90S في النسخ العادية سيرفوات موضعية ولديها نهاية ميكانيكية. لا يمكن جعلها تستمر في الدوران بعد نهايتها بالسوفتوير. إرسال نبضات أبعد لا يعطي دورانًا إضافيًا؛ بل يجعل السيرفو يضغط على النهاية ويكسر التروس أو يسخن.
تمت إزالة الحدود الخاصة بالمجسم فقط، وبقي المجال العام 0°..180° ونطاق النبضة 600..2400us كحماية أساسية.
للدوران المستمر تحتاج سيرفو 360° مستمر أو محرك مع إنكودر، وطريقة التحكم ستكون سرعة/اتجاه وليست زاوية.

محدودية مهمة
------------
السيرفو العادي لا يرسل موضعه الحقيقي للأردوينو. بعد تحريره إذا تحرك يدويًا، النظام لا يعرف مكانه الجديد. لن يرجع وحده لأنه لا توجد إشارة، لكن عند إرسال أمر جديد قد يتحرك مباشرة نحو الهدف الجديد. لمعرفة موقعه الحقيقي تحتاج Feedback Servo أو إنكودر/حساس زاوية.

التشغيل
-------
1) ارفع robot_arm_controller.ino على Arduino UNO.
2) افتح المجلد في VS Code.
3) افتح index.html بواسطة Live Server.
4) استخدم Chrome أو Edge.
5) أغلق Serial Monitor.
6) اتصل بالأردوينو من الصفحة.
7) فعّل الأوامر.
